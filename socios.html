BEM VINDO AO CLUBE DE SÓCIOS!!

✨ Sistema do Clube de Sócios está funcionando 💖


📌 Este sistema foi criação para o Projeto Final da Disciplina de Programação Web
e tem como tema um sistema para Clube de Sócios. Aqui pode-se adicionar um sócio, até 9 dependentes para cada sócio e pagamentos para cada sócio. Pode-se pesquisar um sócio ou dependente e pagementos, além das funcionalidades de editar e excluir para sócio, dependente e pagamento.


📌 O sistema foi feito com HTML, CSS e JS no Front-end e NODE.JS com EXPRESS no Back-end.
